Ben Branch's Engineering Portfolio

Green Houseparty

A Student Innovation Project that monitors soil moisture levels and waters plants while playing music. Built with an ESP32, soil sensor, and speaker. The system uses bytecode to play audio samples.

Text Adventure Treasure Chest

A collaborative project featuring a text-based trivia game. Completing the game activates a servo to open a treasure chest. Developed using an Arduino.

Remote Bot

A remote-controlled car capable of decoding and acting on received signals autonomously. Built with an ESP32 and servo motors.

Drone Project

A self-balancing drone designed with a PID controller for stability. Built with an ESP32, MPU6050, ESCs, and brushless motors. Includes WebSocket-based control.

Larson Scanner

A lighting project creating a smooth scanning effect. This project is inspired by classic sci-fi aesthetics and uses LEDs.

Simon's Lights

A recreation of the Simon memory game with multiple levels, victory, and defeat conditions. Built with LEDs, buttons, and an Arduino.

Drum Machine

A simple drum machine musical instrument using an Arduino. Players can assign pitches and create beats with tactile buttons and a speaker.
