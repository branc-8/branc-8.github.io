Drone Project

This drone project developed alongside Jacob Parks's Live Witness project with his help, and based upon various drone projects on the internet, this drone is designed to balance autonomously using a PID controller. It utilizes an ESP32, an MCU6050, 4 ESCs, a battery, and 4 high speed brushless motors. This drone features a websockets control that can be used from any wifi point.

https://github.com/branc-8/UAT-Drone
